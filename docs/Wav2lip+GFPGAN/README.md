# 音视频唇形匹配技术

仓库地址：[DigitalEraLab/wav2lip-gfpgan](https://github.com/DigitalEraLab/wav2lip-gfpgan)

本项目使用Wav2lip进行唇形匹配以及GFPGAN人脸高清修复。

输入一段音频以及一段视频（或者图片），即可生成高清的声音与画面同步视频。

使用自己的数据集训练有更好的唇形匹配效果。
